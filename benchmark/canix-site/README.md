# Groupe Canix Inc. - Site Benchmark

Projet de site statique pour Groupe Canix Inc.

## Structure du projet

```
benchmark/canix-site/
├── index.html          # Page d'accueil principale
├── css/
│   ├── base.css        # CSS reset et palette de couleurs
│   └── typography.css  # Polices Google Fonts (Playfair Display + Inter)
├── js/
│   └── main.js         # JavaScript principal (À compléter)
└── images/
    └──                 # Images du site (À ajouter)
```

## Palette de couleurs

- **Bleu marine (Primary)**: `#1B2A4A`
- **Or (Accent)**: `#C9A84C`
- **Blanc**: `#FFFFFF`

## Polices

- **Titres**: Playfair Display
- **Corps de texte**: Inter

## Démarrage

1. Ouvrir `index.html` dans un navigateur
2. Personnaliser le contenu selon les besoins
3. Ajouter les images dans le dossier `images/`
4. Ajouter du JavaScript interactif dans `js/main.js`
